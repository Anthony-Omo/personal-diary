# Personal Diary app

## How to install
- Run `npm install`
- Create .env file
- Create supabase db, run `npm run setup-db` after linking your supabase DB Url
- Run `npm run dev` to start the api
- Open index.html to view client side

## Server
- Server side has complete functionality to create, read, update and destroy posts
- Server can retrieve a diary post via id or category

## Client
- There is a form to post and delete posts on the html 
- html also displays all the posts
